# robot-knight

🤖 Knight Robot Game

A small front-end project featuring a CSS-built robot that moves across the screen, blinks its eyes, and can be controlled with interactive buttons.

🧠 Behavior

Once the robot sees you, he starts moving, protecting the castle and dangerously blinking with his eyes.

🚀 Features

🤖 Robot built entirely with CSS
🎞️ Smooth movement animation
👀 Blinking eyes (with toggle on/off)
🎮 Control buttons:
Start / Stop movement
Eyes On / Off
🎨 Game-style UI with styled buttons and layout
📱 Responsive behavior (works on different screen sizes)

🛠️ Technologies Used

HTML
CSS (animations, gradients, shapes)
JavaScript 

🎯 How It Works
-> The robot is created using pure CSS shapes and gradients.
-> Animations are handled with @keyframes.
-> Buttons use JavaScript to:
-> Toggle movement (animation-play-state)
-> Turn eye blinking on/off (by adding/removing a class)

📦 How to Run
1. Clone the repository
2. Open index.html in your browser.
3. Or just run github-pages :)
